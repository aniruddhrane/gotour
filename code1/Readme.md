# Student Management System (Go)

A command-line Student Management System built while learning Go.

## Features

- Add Student
- Remove Student
- Print Student Details
- Update Student Year
- Add Subjects
- List All Students
- Count Students

## Concepts Practiced

- Structs
- Methods
- Pointer Receivers
- Packages
- Slices
- Loops
- Basic CLI
- Go Modules

## Project Structure

student/
- Student struct
- Student methods

manager/
- StudentManager
- Business logic

main.go
- CLI Menu

## Run

```bash
go run .
```
